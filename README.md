## Project Overview
This project presents an interactive and business-oriented Power BI dashboard developed using the Google Play Store dataset. The dashboard provides detailed insights into app performance, user engagement, installs, reviews, ratings, pricing models, and category-wise trends.

The primary objective of this project is to transform raw application data into meaningful business insights that help stakeholders, analysts, and developers understand the key factors influencing app success in the Google Play Store ecosystem.

The dashboard is designed from both a Business Analyst and Stakeholder perspective to support data-driven decision-making.

---

# Business Problem Statement
With millions of applications available on the Google Play Store, businesses and developers need analytical insights to understand:

- Which app categories dominate the market?
- What factors influence app popularity and installs?
- Do higher ratings impact application success?
- Are free apps more successful than paid apps?
- Which applications generate the highest user engagement?
- What strategies can improve app growth and retention?

This dashboard helps answer these business questions using interactive visualizations, KPIs, and category-level analysis.

---

The Power BI dashboard includes multiple interactive visuals and KPI indicators such as:

- Total Applications

# Data Cleaning & Transformation
Several preprocessing and transformation steps were performed in Power BI using Power Query:

- Removed duplicate records
- Handled missing/null values
- Converted installs and reviews into numeric format
- Removed special characters (+, ,, $)
- Standardized pricing and install values
- Corrected inconsistent data formats
- Created calculated columns and DAX measures

These steps improved overall data quality and dashboard accuracy.

# Key Insights

## 1. Free Apps Dominate the Market
The analysis shows that free applications dominate the Google Play Store ecosystem, indicating strong user preference toward freemium business models.

---

## 2. High-Demand Categories
Categories such as:
- Games
- Communication
- Productivity
- Tools
- Social

generate the highest installs and user engagement, reflecting strong market demand.

---

## 3. Ratings vs Installs Relationship
Applications with higher installs generally maintain strong ratings, suggesting a positive relationship between app quality, popularity, and user satisfaction.

---

## 4. User Engagement Analysis
Apps like:
- Instagram
- WhatsApp
- Facebook
- Subway Surfers

receive extremely high review counts and installs, showing strong daily user interaction and engagement.

---

## 5. Paid Apps Performance
Paid apps represent a smaller percentage of total applications, while pricing has minimal impact on ratings and installs.

This suggests that user experience and functionality matter more than pricing.

---

## 6. Category Competition Analysis
Family and Games categories contain the highest number of applications, indicating intense competition and strong user demand.

---

## 7. Success Score Insights
Top-performing applications consistently combine:
- High installs
- Strong ratings
- Large review counts

This indicates that successful applications depend on a balance of:
- User trust
- Market reach
- Customer engagement
- Product quality

## 1. Focus on User Experience
Businesses should improve app usability, performance, and design to maintain strong ratings and increase customer retention.

---

## 2. Adopt a Freemium Model

## 3. Increase User Engagement

## 4. Target High-Growth Categories

- Games
- Productivity
- Communication
- Social

## 5. Maintain Regular Updates
Frequent updates improve:
- User trust
- Application stability

## 6. Use Data-Driven Decision Making
Businesses should continuously monitor installs, reviews, ratings, and engagement metrics to optimize app strategy and market positioning.
