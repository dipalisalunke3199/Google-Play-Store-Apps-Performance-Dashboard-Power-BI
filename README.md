# Google Play Store Apps Performance Dashboard | Power BI

## Project Overview
This project presents an interactive Power BI dashboard built using the Google Play Store dataset to analyze app performance, user engagement, installs, ratings, reviews, pricing, and category-wise trends.

The dashboard helps stakeholders and analysts identify key factors influencing app success through visually driven insights and KPI analysis.

---

# Key Dashboard Features
- KPI Cards (Total Apps, Installs, Reviews, Ratings)
- Top Apps by Installs & Reviews
- Category-wise Performance Analysis
- Free vs Paid App Distribution
- Content Rating Analysis
- Interactive Filters & Slicers
- Success Score Analysis

---

# Key Insights
- Free apps dominate the Play Store ecosystem.
- Games and Communication categories generate the highest installs.
- Apps with strong ratings tend to achieve higher installs.
- Instagram, Subway Surfers, and WhatsApp show high user engagement.
- User experience and app quality significantly impact app success.

---

# Business Recommendations
- Focus on user experience and app performance.
- Adopt freemium business models for better reach.
- Encourage user reviews and engagement.
- Target high-demand categories like Games and Productivity.
- Maintain regular updates for better retention.

# Conclusion
This dashboard transforms raw Google Play Store data into meaningful business insights using interactive visualizations and KPI-driven reporting. It helps businesses, developers, and stakeholders make informed decisions related to app growth, engagement, and market strategy.
